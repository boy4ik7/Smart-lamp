# Смарт-лампа
Компоненти:
- Резистор (200-400 Ом);
- налагоджувальна плата (Arduino Nano);
- кнопка (Сенсорна TTP223);
- адресне світодіодне коло (8-12 світлодіодів);
- модуль датчик освітленості (LM393).
